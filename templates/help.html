User Guide

Follow these simple steps to run the Smart Advertisement Slot Allocation System:

  1. Prepare a dataset text file matching the formatting specifications. You can copy or download sample datasets from this page.
  2. Go to the Upload & Run page, drag & drop your `.txt` file into the upload area (or click to browse).
  3. The system will validate your file in real time. If successful, the Configure Allocation panel unlocks, displaying dataset statistics.
  4. Choose an allocation strategy (Greedy Scheduling or Knapsack Dynamic Programming) and click Optimize & Allocate Slots.
  5. You will be redirected to the results panel, where you can search, sort, filter, and download CSV/PDF reports.

Frequently Asked Questions

The system performs rigorous validation checking for: empty files, missing columns, duplicate advertisement IDs, non-numeric values in numeric columns, and negative durations/budgets/priorities. It also checks that preferred slot values exactly match one of the four configured slots (Morning, Afternoon, Evening, PrimeTime).

No, advertisement IDs must be completely unique. If duplicate IDs are detected, the dataset validation will fail, and the system will notify you of the specific IDs that are repeated.
